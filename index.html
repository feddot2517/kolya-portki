Маски
Yolo shop
Маска для Московского онлайн-магазина Yolo Shop. Эффект содержит неоновую рамку с названием бренда. При моргании появляются зеленые линзы.
Танос
Маска с главным персонажем известного фильма "Мстители". 3D Модель повторяет действия за лицом человека. При записи происходит замена голоса.
Makeup
Маска с 3D ресницами и тенями для блогера Екатерины Колисниченко. Эффект увеличивает размер губ, глаз и уменьшает нос.
Хам-Миллион
Маска-рандом для Омского Бара Хам-Миллион. Над головой человека крутится надпись с названием заведения. В неоновой рамке случайным образом выпадает напиток, на который будет действовать скидка. На лице присутствует эффект светомузыки.
Респиратор
Маска с 3D моделью респиратора и зелеными линзами. При открытии рта появляется зеленый дым.
Моргенштерн
Маска с эффектом трекинга лица человека из видео.
Тест на Вирус
Маска с возможностью отвечать на вопросы. При наклоне головы выбирается ответ: да/нет, в зависимости от выбора определяется итог опроса.
Футболина
Маска с вопросами на футбольную тематику для youtube-шоу "Футболина". Пользователю случайным образом выдается вопрос, на который необходимо ответить в течение 10 секунд. Далее появляется правильный ответ.
Игры
Роза Хутор
Игра с возможностью смены персонажей. Управление осуществляется путем наклона головы. Ведется счет набранных очков и количества жизней.
Гонки
Игра , где нужно управлять автомобилем для того, чтобы объехать другие машины. Управление осуществляется путем наклона головы.
catch food
Игра для Омского кинотеатра "Галактика". Необходимо ловить ртом правильные предметы, в обратном случае будет отниматься одна жизнь. Игра заканчивается когда человек поймал 3 отрицательных объекта. Ведётся счет лучшего рекорда.
Марио
Игра, оформленная в стиле известной игры Марио. Управление персонажем осуществляется путем моргания. Игра заканчивается, если врезаться в препятствие.
Попробуй удержать
Игра для футбольной группы footbline. Цель игры удержать мяч, балансируя им на голове. Если объект падает, то происходит перезапуск по касанию экрана.
Омгту
Игра для Омского Государственного Технического Университета. С помощью наклонов головы можно управлять автомобилем и объезжать препятствия.
Пресеты
Teal & Orange

Kira-Kira

PolaroId
Пресет с полароидной рамкой. По касанию появляются анимированные пылинки.
Labels
Пресет с ретушью на двоих и различными фразами желтого цвета, которые меняются по касанию экрана.
360 градусов
Винтажный пресет с фоном гор и озера 360 градусов, которые меняются по касанию экрана.
Shadow
